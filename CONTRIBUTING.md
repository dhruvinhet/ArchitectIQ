# Contributing to ArchitectIQ

Thank you for your interest in improving ArchitectIQ.

## Development Setup

1. Install dependencies:

```bash
npm install
```

2. Compile extension source:

```bash
npm run compile
```

3. Launch Extension Development Host in VS Code (`F5`).

## Contribution Guidelines

- Keep changes focused and small.
- Follow existing TypeScript style and naming conventions.
- Add or update tests when behavior changes.
- Update `README.md` and `CHANGELOG.md` for user-visible features.
- Avoid introducing telemetry or remote data transfer without explicit documentation.

## Pull Request Checklist

- [ ] Code compiles successfully (`npm run compile`)
- [ ] Changes are documented in `CHANGELOG.md` when applicable
- [ ] User-facing docs are updated
- [ ] Security and privacy implications are considered

## Code of Conduct

Be respectful, constructive, and professional in all interactions.
